File Storage Location

All test data and generated PDF reports are automatically organized and stored on the shared network drive. You can find the root directory for all sales order records at:

T:\OTSDocs\OTS DLS Chart Recordings

How Files are Organized

The system follows a strict hierarchical structure to ensure every test is easily traceable:

  • 1. Sales Order: The top-level folder representing the project.
  • 2. Item Number: Sub-folders for specific items within that order.
  • 3. Valve Folder: Individual folders for each valve tested.
  • 4. Data & PDF: Each valve folder contains raw CSV data/JSON metadata and the final generated PDF reports.

The Batch Serial

All tests performed simultaneously are linked by a unique timestamp serial, such as 18-02-2026_14-03-00. This ensures that even if multiple valves are tested in one go, their results stay grouped together.
